# Проект Neutrino

Данный репозиторий относится к проекту [Neutrino](https://github.com/agnostic-t/neutrino-core), является базовой реализацией модуля `transport`.

## Содержание

На данный момент содержит реализации:

- [tcp](./basic/tcp/tcp.go): базовый транспорт через установление TCP соединений. Эта реализация является примером для создания других транспортов (например HTTP)
- [html](./basic/html/html.go): базовый транспорт через установление HTTP сессии. Реализация сервера мимикрирует под сервер Nginx если сервер используется не приложением VPN

Планируется добавление:

- `udp`: базовый транспорт через установление UDP (session-like) соединений. Будет полезен при использовании мультиплексора `quic`
- `https`: транспорт, который иммитирует (или использует) HTTPS запросы
- `ws`: транспорт, который использует технологию WebSocket для передачи данных
